# Minimal Omarchy Installation Guide (Base Layer)

This guide follows the "Maslow Hierarchy of Needs" for a fresh Omarchy (Arch/Hyprland) installation, streamlined for speed and your latest tool preferences (Bun, dotfile linking).

## Phase 1: Physiological (System & Basic Tools)
*Goal: A working system with internet, terminal, and your primary runtime.*

1. **Install Omarchy** from the USB ISO.
2. **Update System:**
   ```bash
   sudo pacman -Syu
   ```
3. **Primary Runtime (Bun):**
   ```bash
   curl -fsSL https://bun.sh/install | bash
   # Restart terminal or source ~/.bashrc
   ```
4. **Shell Tools (Standard Arch/AUR):**
   ```bash
   sudo pacman -S fzf ripgrep fd bat github-cli pandoc meld
   # Note: Standard 'fd' and 'bat' on Arch, no 'fdfind' or 'batcat' needed.
   ```

## Phase 2: Safety (Identity & Dotfiles)
*Goal: Establish your digital identity and bring in your personal configurations.*

1. **Restore SSH Identity:**
   *Copy your `id_rsa` and `id_rsa.pub` from your secure USB/backup to `~/.ssh`.*
   ```bash
   chmod 600 ~/.ssh/id_rsa
   chmod 644 ~/.ssh/id_rsa.pub
   ```
2. **Clone Dotfiles:**
   ```bash
   git clone git@github.com:ianchanning/dotfile.git ~/dotfile
   ```
3. **Link Your Configuration:**
   ```bash
   # WezTerm
   mkdir -p ~/.config/wezterm
   ln -s ~/dotfile/wezterm/wezterm.lua ~/.config/wezterm/wezterm.lua

   # Neovim (Omarchy ships with Nvim, but we link our config)
   mkdir -p ~/.config/nvim
   ln -s ~/dotfile/nvim ~/.config/nvim

   # Tmux
   ln -s ~/dotfile/tmux.conf ~/.tmux.conf
   ```
4. **Backups:**
   - **System**: Use `omarchy-snapshot create` for system-level safety.
   - **Personal Files**: `flatpak install flathub org.gnome.DejaDup` (Restores `~/.var` for app settings).

## Phase 3: Love & Belonging (Communication)
*Goal: Reconnect with your team and sync your work.*

1. **Core Communication (Omarchy Menu / Flatpaks):**
   *Most of these are available via Super + Alt + Space > Install > Package/AUR.*
   - **1Password**: `yay -S 1password 1password-cli` (Critical for secrets).
   - **MEGA**: `yay -S megasync`.

## Phase 4: Esteem (Professional Productivity)
*Goal: Your high-level dev tools and aesthetic foundation.*

1. **Fonts (The Aesthetic Foundation):**
   ```bash
   yay -S ttf-departure-mono ttf-iosevka
   # For LMMono (Latin Modern Mono), copy from your backup to ~/.local/share/fonts
   fc-cache -fv
   ```
2. **Developer Tools:**
   - **Docker**: `sudo pacman -S docker docker-compose`.
   - **AI Agents**: Omarchy ships with `Claude Code` (`cx`) and `OpenCode` (`c`).
   - **Gemini CLI**: `bun install -g @google/gemini-cli`.
   - **Joplin (via Bun)**: `bun install -g joplin`.

## Phase 5: Self-Actualization (The Omarchy Way)
*Goal: Final polish and system-native tweaks.*

1. **Clean up `.bashrc`**:
   - Standardize aliases: Replace `alias fd='fdfind'` with `alias fd='fd'`.
   - Standardize aliases: Replace `alias bat='batcat'` with `alias bat='bat'`.
2. **Hyprland Native Controls**:
   - `Super + Space`: Application Launcher.
   - `Super + Alt + Space`: Omarchy Control Menu.
   - `Super + Return`: Terminal (Automatically uses WezTerm if set in Defaults).
